# Week 1: LLM Fundamentals

## 🎯 Goal
Understand how LLMs work — tokens, embeddings, prompts — and be able to call an LLM confidently.

## ⏱️ Time Budget
- Target: 7-8 hours
- Actual: ___

## 📖 Learning Tasks
- [ ] Watch / read: [Microsoft Learn — Understand Generative AI](https://learn.microsoft.com/en-us/training/modules/fundamentals-generative-ai/) (30 min)
- [ ] Read: [OpenAI Prompt Engineering Guide](https://platform.openai.com/docs/guides/prompt-engineering)
- [ ] Read: [Microsoft Learn — Understand Embeddings](https://learn.microsoft.com/en-us/training/modules/understand-embeddings/) (45 min)
- [ ] Understand: tokens, temperature, top-p, context window
- [ ] Understand: cosine similarity & vector search basics

## 🛠️ Hands-On
- [ ] Count tokens in 3 different prompts using the [tokenizer](https://platform.openai.com/tokenizer)
- [ ] Write 3 versions of a prompt (vague, specific, few-shot) — compare results
- [ ] Try chain-of-thought ("Think step by step…") on a math problem

## ✅ Success Criteria
- [ ] Can explain in plain English: "What is an LLM?"
- [ ] Can explain why embeddings matter for RAG
- [ ] Can identify a bad prompt vs a good prompt

## 📦 Deliverable
- [ ] `notes/week-01-llm-fundamentals.md` with key concepts
