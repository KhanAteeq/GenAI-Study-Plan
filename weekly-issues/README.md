# Weekly Issue Bodies

Pre-written GitHub Issue bodies for each of the 12 weeks. Workflow:

1. Open https://github.com/KhanAteeq/GenAI-Study-Plan/issues/new/choose
2. Pick the **Weekly Task** template (or just "Open a blank issue")
3. Title it: `Week X: <topic>` (matches file name here)
4. Copy the contents of the matching `week-XX-*.md` file into the issue body
5. Assign it to the right milestone (Month 1 / 2 / 3) and add labels
6. Check off boxes as you progress; close the issue when done

**Tip**: Create the 3 milestones first in GitHub: `Month 1`, `Month 2`, `Month 3`.
